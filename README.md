# Divya Patel - Portfolio

Live site: **https://nileshshukla.github.io/divyabenpatel/**

## Want to update the site?

👉 **Read [EDITING.md](EDITING.md)** - everything editable lives in one file,
[`content.js`](content.js). Edit it right here on GitHub (pencil icon), commit,
and the site updates itself in about a minute. Photos go in the
[`images/`](images) folder.
